# odin-recipes
odin-recipes 

This is a website that is used to find the recipes of food.It contains a home page having list of food items with corresponding links. These links navigates to corresponding web pages that contains the brief description of the food item with the ingredients and the steps to prepare the food easily.

This is a project that helps to grasp the concept of html and the usage of different html elements such as images, headings, links etc.
